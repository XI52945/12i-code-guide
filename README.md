## 👋 写在前面

你好，我是 12i。

在这个 AI 爆发的时代，很多人都在问：“我还需不需要学习编程？”

我的答案是：需要，但方式变了。

过去，我们需要像背字典一样死记硬背每一行晦涩的语法；现在，我们只需要像指挥家一样，理解逻辑，驾驭 AI，让代码随着思维流淌。这就是我所提倡的 "Vibe Coding" —— 一种注重感觉、心流与创造力的编程新范式。

12i Code 不是一本厚重的教科书，也不是一个冰冷的文档库。它是为你准备的一个随时待命的辅助器。

当你迷茫于浩如烟海的技术名词时，它为你指路；
当你困惑于莫名其妙的报错红字时，它为你解惑；
当你想要快速实现一个想法时，它给你最趁手的兵器。

我希望这个项目能成为你推开代码世界大门的那把钥匙。别害怕犯错，别担心学不会，保持好奇，享受创造。

让我们一起，Happy Vibe Coding! ✨

## 🛠️ 本地运行

如果你想在本地运行这个项目，或者参与改进：

1.  克隆仓库
    ```bash
    git clone https://github.com/XI52945/12i-code-guide.git
    cd 12i-code-guide
    ```

2.  安装依赖
    ```bash
    npm install
    ```

3.  启动项目
    ```bash
    npm run web
    # 或者 npm run android / npm run ios
    ```
